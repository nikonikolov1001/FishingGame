🎣 Fishing Boss Adventure

Fishing Boss Adventure is a 2D browser-based fishing game where players catch fish, feed powerful sea creatures, and travel across the ocean to face increasingly challenging bosses.

The game combines simple fishing mechanics with progression and boss encounters, creating a fun and engaging gameplay loop.

🐟 Gameplay
Cast your hook into the water and catch fish
Collect fish and use them as food for massive sea bosses
Each boss requires a certain amount of fish to be satisfied
Once a boss is fed, you travel to a new area to face the next one
Every new boss requires more fish than the previous
🐉 Boss Progression
🐢 Sea Turtle – Easy starter boss
🦈 Reef Shark – Faster and more demanding
🐋 Blue Whale – Huge and requires a lot of fish
🚤 Features
Smooth fishing mechanics (cast, move, catch)
Boss feeding system with progression
Travel system between areas (no instant teleporting)
Increasing difficulty and scaling requirements
Pixel-style visuals inspired by arcade fishing games
🎮 Controls
A / D – Move hook left / right
Mouse Click – Cast hook
F – Feed collected fish to boss
ESC – Pause (if implemented)
🛠️ Technologies
HTML5 Canvas
JavaScript (Vanilla)
CSS
🚀 How to Run
Download or clone the repository
Open index.html in your browser
Start fishing! 🎣
